Create an empty line with only one chevron `>` before starting to add pictures again, and it will create a new strip.

```md
> [!film-strip|wide]
> ![[Image1.jpg]]
> ![[Image2.jpg]]
> ![[Image3.jpg]]
> ![[Image4.jpg]]
> ![[Image5.jpg]]
> ![[Image6.jpg]]
> ![[Image7.jpg]]
> ![[Image8.jpg]]
> ![[Image9.jpg]]
> ![[Image10.jpg]]
```
